Project

Since World of Warcraft armory changed from team-based to player-based style many players missed old armory. So I've decided to give it a try and implement some data mining and machine learning to bring old armory back and throw some minor features on top. Hope you find it helpful.

FotM chart is based on the number of games teams played in those setups, i.e. if a particular setup has 10% representation shown, it was seen in 10% of all games among top players.

How it works?

There's a scanner running 24/7 polling top 1000 players from Blizzard API. Whenever it sees players that changed ratings it calculates diffs from their previous values.

The diffs are then fed to clustering algorithm that is a variation of k-means to try and divide players into the teams of 3.
